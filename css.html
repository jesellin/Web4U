Ejemplos CSS — 5 propiedades

  • display
  • flexbox
  • grid

    • EN

display

Define cómo se renderiza un elemento en el flujo de la página.

.box{display:flex;align-items:center;justify-content:center;}

Uso práctico: Centrar contenido y crear layouts flexibles. W3Schools

flexbox

Modelo de caja para diseñar en una dimensión (fila o columna).

.container{display:flex;gap:8px}
.container > *{flex:1}

Uso práctico: Barras de herramientas o listas horizontales. W3Schools

grid

Sistema bidimensional con filas y columnas.

.grid{display:grid;grid-template-columns:1fr 2fr;gap:12px}

Uso práctico: Diseños complejos tipo dashboard. W3Schools

color / background

Control de color de texto y fondos.

.lead{color:#1155cc;background:#f0f7ff;padding:8px}

Uso práctico: Jerarquía visual y contraste. W3Schools

margin / padding

Espacios exterior e interior para separar elementos.

.card{margin:12px;padding:14px;border-radius:8px}

Uso práctico: Espaciado coherente en interfaces. W3Schools
