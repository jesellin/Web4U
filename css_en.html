CSS Examples — 5 properties

  • display
  • flexbox
  • grid
  • color / background
  • margin / padding

display

Defines how an element is rendered in the page flow.

.box{display:flex;align-items:center;justify-content:center;}

Practical use: Centering content and creating flexible layouts. W3Schools

flexbox

Box model for designing in one dimension (row or column).

.container{display:flex;gap:8px}
.container > *{flex:1}

Practical use: Toolbars or horizontal lists. W3Schools

grid

Two-dimensional system with rows and columns.

.grid{display:grid;grid-template-columns:1fr 2fr;gap:12px}

Practical use: Complex layouts like dashboards. W3Schools

color / background

Control of text and background colors.

.lead{color:#1155cc;background:#f0f7ff;padding:8px}

Practical use: Visual hierarchy and contrast. W3Schools

margin / padding

Outer and inner spacing to separate elements.

.card{margin:12px;padding:14px;border-radius:8px}

Practical use: Consistent spacing in interfaces. W3Schools
