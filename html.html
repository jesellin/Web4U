Ejemplos HTML — 5 etiquetas

  • <header>
  • <nav>
  • <article>
  • <form>
  • <table>

<header>

Elemento semántico que agrupa la cabecera de una página o sección.

<header>
  <h1>Título del sitio</h1>
</header>

Uso práctico: Agrupar el logo y la navegación principal. W3Schools

<nav>

Contenedor para enlaces de navegación principales.

<nav>
  <a href="/">Inicio</a>
  <a href="/about">Sobre mi</a>
</nav>

Uso práctico: Agrupar menús y facilitar el acceso. W3Schools

<article>

Contenido autocontenido que puede repartirse independientemente.

<article>
  <h2>Noticia</h2>
  <p>Texto...</p>
</article>

Uso práctico: Entradas de blog, noticias o widgets independientes. W3Schools

<form>

Formulario para recopilar datos del usuario.

<form action="/enviar" method="post">
  <label>Nombre <input name="nombre" required></label>
  <button>Enviar</button>
</form>

Uso práctico: Recolectar datos (contacto, login). W3Schools

<table>

Representa datos en forma tabular.

<table>
  <caption>Horario</caption>
  <thead><tr><th>Hora</th><th>Asignatura</th></tr></thead>
  <tbody><tr><td>9:00</td><td>Matemáticas</td></tr></tbody>
</table>

Uso práctico: Mostrar datos estructurados. W3Schools
