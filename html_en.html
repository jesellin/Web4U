<header>

Semantic element that groups the header of a page or section.

<header>
  <h1>Site Title</h1>
</header>

Practical use: Grouping the logo and main navigation. W3Schools

<nav>

Container for main navigation links.

<nav>
  <a href="/">Home</a>
  <a href="/about">About</a>
</nav>

Practical use: Grouping menus for easy access. W3Schools

<article>

Self-contained content that can be distributed independently.

<article>
  <h2>News Item</h2>
  <p>Text...</p>
</article>

Practical use: Blog posts, news, or independent widgets. W3Schools

<form>

Form to collect user data.

<form action="/send" method="post">
  <label>Name <input name="name" required></label>
  <button>Send</button>
</form>

Practical use: Data collection (contact, login). W3Schools

<table>

Represents tabular data.

<table>
  <caption>Schedule</caption>
  <thead><tr><th>Time</th><th>Subject</th></tr></thead>
  <tbody><tr><td>9:00</td><td>Math</td></tr></tbody>
</table>

Practical use: Displaying structured data. W3Schools
